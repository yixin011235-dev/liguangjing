## 🔍 我的自动化能力

当你发送「部署博客」时，我会：

### 1. **检测变化**
- 检查 `source/_posts/` 目录
- 识别新增或修改的文章
- 显示即将部署的内容

### 2. **生成网站**
```bash
hexo clean
hexo generate
```
- 清理旧文件
- 生成新的静态网站
- 处理所有文章和资源

### 3. **部署到服务器**
```bash
.\deploy.bat
```
- 使用 rsync 同步文件
- 设置正确的文件权限
- 重启 Nginx 服务
- 全程自动化，无需人工干预

### 4. **备份到 GitHub**
```bash
git add .
git commit -m "更新博客"
git push origin main
```
- 自动提交所有更改
- 推送到 GitHub 仓库
- 保留完整的版本历史

### 5. **反馈结果**
- 显示部署状态
- 提供访问链接
- 提示可能的问题
