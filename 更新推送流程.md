## 🎯 工作原理

当你说"部署博客"时，我执行：

```powershell
cd d:\my-blog
npx hexo clean && npx hexo generate
scp -r .\public\* ubuntu@175.178.189.55:/var/www/hexo/
ssh ubuntu@175.178.189.55 "sudo systemctl restart nginx"
git add . && git commit -m "更新" && git push
```
