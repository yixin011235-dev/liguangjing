## 六、已修改的文件清单

以下文件相对于原版 hipaper 主题做了定制修改：

### 1. themes/hipaper/layout/page.ejs
- **修改原因**：原版 page.ejs 不支持 type 为 tags/categories 的页面路由，导致分类和标签页面显示 404
- **修改内容**：添加了按 page.type 判断的条件路由

```ejs
<% if (page.type === 'tags') { %>
  <%- partial('tags') %>
<% } else if (page.type === 'categories') { %>
  <%- partial('categories') %>
<% } else { %>
  <%- partial('_partial/article', {post: page, index: false}) %>
<% } %>
```

### 2. themes/hipaper/layout/_widget/social.ejs
- **修改原因**：WeChat 的 URL 为空时，点击图标会跳转到首页；需要支持微信二维码弹窗
- **修改内容**：当 key 为 WeChat 且 URL 为空时，使用 javascript:void(0) 并显示二维码弹窗（图片路径：css/images/weixinhao.jpg）

### 3. themes/hipaper/layout/_partial/footer.ejs
- **修改原因**：需要在页脚添加 ICP 备案号
- **修改内容**：在 site-info 中添加了"鄂ICP备2025104835号"，链接指向 https://beian.miit.gov.cn/，样式为灰色无下划线

### 4. themes/hipaper/source/css/_partial/sidebar-aside.styl
- **修改原因**：配合微信二维码弹窗功能
- **修改内容**：在文件末尾添加了 .wechat-qrcode 及 .qrcode-popup 的完整样式（包括悬浮显示、定位、阴影、箭头指示器等）

### 5. themes/hipaper/_config.yml
- **avatar.width/height**：从 200 改为 67
- **social.WeChat**：设置为空（触发二维码弹窗）
- **social.Github**：设置为 https://github.com/yixin011235-dev
- **description 相关**：见站点 _config.yml

### 6. _config.yml（站点配置）
- 各项站点信息已配置完毕（标题、描述、关键词、语言等）

### 7. .gitignore（项目根目录）
- 排除：.DS_Store, Thumbs.db, db.json, *.log, node_modules/, public/, .deploy*/, .multiconfig.yml

## 九、部署流程

### 方式一：通过 hexo deploy（推荐）
```bash
hexo clean
hexo generate
hexo deploy
```
deployer-git 会自动推送到 root@175.178.189.55:/root/hexo.git

### 方式二：通过 deploy.bat 脚本
直接运行 deploy.bat，脚本会：
1. 清理并生成静态文件
2. 通过 SCP 上传到服务器 /var/www/hexo

### 服务器端 Nginx 配置
- 域名：liguangjing.top
- 监听 80 和 443 端口
- SSL 证书路径：/etc/nginx/ssl/（从 liguangjing.top_nginx/ 目录上传）
- 网站根目录：/var/www/hexo

## 十、SSH 密钥配置

本机已配置 SSH 密钥用于免密码连接服务器：
- 公钥已添加到服务器 /root/.ssh/authorized_keys
- 部署到服务器时无需输入密码

如果另一台电脑也需要部署到服务器，需要：
1. 生成 SSH 密钥：`ssh-keygen -t ed25519 -C "你的邮箱"`
2. 将公钥添加到服务器：`ssh-copy-id root@175.178.189.55`
